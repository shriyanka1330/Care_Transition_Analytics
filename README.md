# Care Transition Efficiency & Placement Outcome Analytics

A data-driven analytics dashboard for monitoring **care transition efficiency**, **placement outcomes**, and **operational performance** using **Python**, **Streamlit**, **Pandas**, and **Plotly**.

---

## Project Overview

This project analyzes operational care transition data to measure workflow efficiency, monitor placement outcomes, and identify potential bottlenecks through interactive dashboards and visual analytics.

The application transforms raw operational data into actionable insights using statistical analysis, Key Performance Indicators (KPIs), and dynamic visualizations.

---

## Features

* Interactive Streamlit dashboard
* KPI summary cards
* Time-series trend analysis
* Correlation analysis
* Sankey flow visualization
* Cumulative flow analysis
* Data filtering
* Operational performance metrics
* Interactive Plotly charts
